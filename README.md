# svelte-prettier-eslint-template

Repository with my configuration for Prettier and ESLint for Svelte projects

# Setup

Create your Svelte project:

```
npx degit sveltejs/template my-svelte-project
cd my-svelte-project

npm install
npm run dev
```

Install dependencies required to make ESLint and Prettier work with Svelte (Assuming that you have `Svelte Beta` plugin for VSCode)

```
npx degit icebr9/svelte-prettier-eslint-template .
```
